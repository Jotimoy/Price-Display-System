Hardware Components

📡 ESP8266 / ESP32

Main IoT microcontroller responsible for WiFi communication and data fetching from the server.

🖥 LED Matrix Display

Displays real-time commodity prices in digital format in market areas.

🔌 Power Supply Unit

Provides stable voltage supply for continuous operation.

📶 WiFi Router

Ensures internet connectivity for real-time centralized updates.

Software Architecture

💻 Admin Dashboard

Web-based system for updating daily commodity prices.

🗄 Cloud Database

Stores updated price data securely (Firebase / MySQL).

🔗 REST API

Handles communication between server and IoT devices.

System Flow Diagram

Admin Panel → Cloud Server → Internet → ESP8266 → LED Display


The administrator updates prices from the dashboard. Data is stored in the cloud database and automatically fetched by connected IoT boards via WiFi.
