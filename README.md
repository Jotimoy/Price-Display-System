# Smart Market Price Display System

This is a research project website for the IoT-based Centralized Digital Market Price Display System.

## Project Overview

The system proposes a scalable and low-cost digital board system that enables centralized real-time commodity price updates across multiple market locations using IoT technology.

## Features

- **Centralized Admin Dashboard**: Web-based control panel for updating prices
- **IoT Integration**: ESP8266/ESP32 microcontrollers for real-time updates
- **LED Display Panels**: High-brightness displays for market environments
- **Cloud Synchronization**: Firebase/MySQL for data storage
- **Responsive Website**: Project presentation and documentation

## Technology Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Python Flask
- Database: Firebase / MySQL
- Hardware: ESP8266, P10 LED Matrix, SMPS
