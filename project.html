Research Methodology

This page presents the core steps our project followed, from requirement analysis to implementation and evaluation. The design is built to make the process easy to understand at a glance.

1. Requirement Analysis

We studied existing market display systems and identified the need for a centralized, low-cost, and real-time price update platform tailored for local market environments.

2. System Design

The architecture was split into three layers: the admin dashboard, cloud communication, and the IoT-enabled display board for market screens.

3. Prototyping

Hardware prototypes using ESP32/ESP8266, LED display units, and sensors were built to verify reliable price transmission and board update behavior.

4. Validation

Real-world functionality was tested by updating prices centrally and confirming that market boards synchronized instantly while maintaining readability and performance.

System Block Diagram

The diagram below shows the message flow from the admin panel through WiFi to the ESP32 microcontroller and finally to the display board.

Architecture highlights

The project uses an admin panel that sends price information via WiFi to an ESP32 microcontroller. The ESP32 processes updates and pushes them to the LCD display board in real time.

Key components

Admin interface, WiFi communication link, ESP32 controller, and LCD market display.

Benefits

Reliable wireless delivery, low-cost hardware, and instant price updates for market boards.

Hardware Wiring Diagram

Detailed connection flow between the ESP32 module, LED display panel, and power source used in the real implementation.

ESP32 wiring details

This hardware diagram illustrates how the ESP32 is connected to the LED display panel and the power source. The ESP32 controls the display through GPIO signal lines while the power module supplies stable 5V and ground to both devices.

Signal connections

Data and control pins from the ESP32 are mapped to the display module for reliable refresh and update timing.

Power distribution

The power supply feeds both the ESP32 and the LED display, ensuring the display receives stable voltage during full-panel updates.

Impact & Benefits

These research outcomes show how the Smart Market Price Board improves market transparency, reduces price update delays, and creates a reliable information system.

Real-time updates

The system instantly broadcasts new commodity prices to all connected display boards from one central source.

Cost-effective

Designed for low-budget deployment using affordable IoT components and lightweight communication protocols.

Scalable

The architecture supports multiple market boards with a single backend, enabling future expansion.

User-friendly

Simple admin controls and an intuitive display make the system easy to manage for market operators.
