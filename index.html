Design and Implementation of an IoT-Based Smart Market Price Display System for Real-Time Data Synchronization

Centralized Digital Market Price Display System

IoT Based Smart Market Information System

This research project proposes a scalable and low-cost digital board system that enables centralized real-time commodity price updates across multiple market locations using IoT technology.

📄 Download Proposal 🔍 View Components

Problem Statement

Traditional market price lists are manually updated and often inconsistent. There is no centralized system to control and synchronize daily commodity prices, leading to confusion and lack of transparency.

Proposed Solution

Our system introduces a cloud-connected IoT-based digital display board. Prices are updated from a central admin dashboard and automatically synchronized across all connected market boards via WiFi.

Project Objectives

✔ Develop IoT-based digital display hardware
✔ Implement centralized admin dashboard
✔ Ensure real-time synchronization
✔ Provide scalable low-cost solution

Technology Stack

ESP8266 / ESP32
LED Matrix Display
Python Flask Backend
Firebase / MySQL
Netlify Hosting

System Workflow

1️⃣ Admin Updates Price
2️⃣ Data Stored in Cloud
3️⃣ ESP Fetches Data
4️⃣ Digital Board Updates
